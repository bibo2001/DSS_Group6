## DSS_project (.ipynb)
User 輸入土壤資訊，使用 Random Forests 預測最適的種植作物推薦給 User，提供農業社區一個先進的工具，幫助農民在開始種植作物之前做出明智的決定，來提高生產力、節省資源並實現可持續的農業實踐。
